## 2. 指针形参和指针引用形参

```c++
void fun(int*);		//指针形参
void fun(int* &);	//指针引用形参
```

当只需要在函数中改变指针指向的对象的值时，两者都可以使用。

当需要改变指针指向的地址时，只能使用指针引用形参

因为如果不使用`&`，那么在函数中会自动调用指针的构造函数，构造一个临时指针，这个指针和传进来的指针指向同一个位置。我们可以通过这个临时指针改变该位置的数值，但是如果把临时指针指向另一个位置时，并不会改变传进来的指针指向的位置。
